Guide d’installation
Prérequis

Avant d’installer le projet, assurez-vous d’avoir les outils suivants installés sur votre machine :

Node.js (version 18 ou plus)
npm
Git

Installation du projet

Télécharger ou cloner le repository du projet.

Ouvrir le dossier du projet dans votre éditeur (VS Code par exemple).

Installer les dépendances avec la commande :

npm install

Lancer le projet en local

Pour démarrer le serveur de développement :

npm run dev

Ensuite ouvrir le navigateur et aller sur :

http://localhost:3000

Structure importante du projet

Les pages principales du projet sont situées dans le dossier :

app/

Les images utilisées par l'application doivent être placées dans :

public/images/

Exemple d’images

public/images/session-logo.png
public/images/loading-logo.svg
public/images/intro-bg.jpeg
