# Enchanted Tools – Technical Architecture

## Project Type
Progressive Web Application (PWA)

This project is designed as a Progressive Web Application in order to provide a lightweight, accessible and scalable digital experience.

## Tech Stack

### Frontend
Next.js (React)

### Backend
Next.js API Routes

### Styling
Tailwind CSS

### Hosting
Vercel

### Database
PostgreSQL (planned for future implementation)

## Architecture

The platform follows a simple and scalable architecture:

Utilisateur  
↓  
Frontend (Next.js)  
↓  
API Routes  
↓  
Database

## Features

- Landing page
- Admin dashboard
- API-based architecture
- Scalable structure
- Future integration of dynamic content and user interactions

## Technical Choices

The technical stack was selected for the following reasons:

- **Next.js**: modern React framework, performant, scalable, suitable for web applications
- **Tailwind CSS**: fast UI development and lightweight styling
- **Vercel**: optimized hosting for Next.js with simple deployment
- **PostgreSQL**: reliable relational database for future data management

## Feasibility

The proposed solution is technically feasible because:

- it is based on a modern and standard web stack
- the architecture is modular and easy to extend
- the prototype can evolve progressively into a full product
- the deployment process is simple and reliable

## Hosting

The project is deployed on Vercel.

Advantages:
- easy deployment
- optimized for Next.js
- CDN distribution
- scalable hosting infrastructure

## Eco Design

The project integrates eco-design principles through:

- optimized frontend structure
- reduced unnecessary network requests
- lightweight architecture
- CDN hosting for better performance and reduced server load
