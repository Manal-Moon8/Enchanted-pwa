Guide Admin
Interface Admin

Le projet contient une interface admin simple permettant d’organiser l’expérience visiteur et de préparer la gestion future de l’application.

Accès à l’interface admin

En local, l’interface admin est accessible via :

http://localhost:3000/admin

Fonction de l’admin

L’interface admin peut servir à :

gérer les codes de session
organiser les modules de l’expérience
configurer les zones de l’espace
préparer l’évolution future du projet

Évolutions possibles

Dans une version future, l’admin pourra inclure :

authentification des administrateurs
gestion de contenu dynamique
statistiques visiteurs
gestion avancée des modules
