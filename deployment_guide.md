Guide de déploiement
Plateformes utilisées

Le projet est déployé en utilisant les outils suivants :

GitHub
Vercel

Étapes de déploiement

Envoyer le projet sur GitHub.

Aller sur le site :

vercel.com

Créer un nouveau projet en cliquant sur :

New Project

Sélectionner le repository GitHub contenant le projet.

Vérifier que le framework détecté est :

Next.js

Cliquer sur :

Deploy

Déploiement automatique

Chaque fois que vous faites un push sur la branche principale du repository, Vercel va automatiquement redéployer le projet.

URL finale

Une fois déployé, le projet sera accessible via une URL publique par exemple :

https://enchanted-pwa.vercel.app
